# How to run sqRoot.hs

1. To run testcases:
   ghc -o sqRoot sqRoot.hs
   ./sqRoot
2. To call the function
   ghci
   :l "sqRoot.hs"
   sqroot (x)
3. Input Constraint: It takes only positive decimal numbers.
